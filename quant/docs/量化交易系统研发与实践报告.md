## 三、研发阶段 (R&D)

### 3.1 期货策略研发

#### 3.1.1 3连K策略 (Strategy v3)

**策略原理：**

```
放量K线出现 → 等待2根K线确认 → 第3根K线突破买入
     │              │              │
     ▼              ▼              ▼
  量>3倍        验证趋势        趋势确认
  20日均量      继续上涨        执行买入
```

**核心参数：**
- `lookback_bars`: 15 (回看K线数)
- `volume_multiplier`: 3 (放量倍数)
- 止盈止损: 半仓1:1，全仓2:1

#### 3.1.2 主力合约筛选

**筛选逻辑：**

```python
# 基于成交量最大的合约作为主力合约
# 筛选范围: 6个月内到期合约
# 交易所: CFFEX, SHFE, DCE, CZCE, INE, GFEX
```

**筛选结果：**

| CFFEX | 21 | IF, IC, IH, IM, T, TF, TS, TU |

| DCE | 11 | a, b, c, m, y, p, j, l, v, eg, pg |
| CZCE | 9 | c, cs, rm, ma, ta, zc, pf, jr, sr |
| INE | 3 | sc, nr, bc |

**总计: 54个主力合约**

### 3.2 A股选股策略研发

#### 3.2.1 选股条件

| 条件 | 说明 |
|------|------|
| 近15日涨幅 | < 15% |
| 距60日高点 | < 20% |
| 量比 | > 1.5倍 |
| 缩量条件 | 前2日中有1日成交量 < 20日均值 |

#### 3.2.2 剔除条件

| 条件 | 说明 |
|------|------|
| 创业板 | 300xxx |
| 科创板 | 688xxx |
| ST股票 | *ST, ST |
| 北交所 | BJxxx |
| 市值 | < 50亿元 |

#### 3.2.3 卖点分析

- **5%止盈**: 买入后涨幅达到5%止盈
- **5%止损**: 买入后跌幅达到5%止损
- **30天强制平仓**: 持有30天后强制卖出

**配置:**
```json
{
    "time_frame": "1min",
    "strategy": "3consecutive_k",
    "check_interval": 60,
    "reported_breakouts": []
}
```

**运行状态:**
- 实时获取1分钟K线数据
